## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov

## Additional Setup
## Task API
$ npm i --save class-validator class-transformer
$ npm i @nestjs/mongoose mongoose
$ npm run start:dev

## Task Front
$ npm install -D tailwindcss postcss autoprefixer
$ npx tailwindcss init -p
$ npm i --save class-validator class-transformer
$ npm install tailwindcss
$ npm run dev
```
